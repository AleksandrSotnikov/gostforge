# Каталог проверок

Полный набор проверок зрелой системы: ~75 пунктов в 13 категориях. На MVP включаются 12–15 самых частых; остальные подключаются постепенно.

## Кодирование

Каждая проверка имеет код вида `БУКВА.NN`:
- `F` — Formatting (страница)
- `T` — Text (основной текст)
- `S` — Structure (структура работы)
- `H` — Headings (заголовки)
- `I` — Images (рисунки)
- `B` — Tables (таблицы)
- `M` — Math (формулы)
- `L` — Lists (списки)
- `R` — References (литература)
- `C` — Cross-references (перекрёстные ссылки)
- `A` — Abbreviations
- `P` — Appendices (приложения)
- `K` — Headers/footers (колонтитулы)
- `V` — Volume metrics
- `X` — Extra (стиль, лингвистика)

**Серьёзность:** `error` (нарушение ГОСТа/профиля, обязательно к исправлению), `warning` (вероятная проблема), `info` (предложение по улучшению).

**Статус:** `✓` — реализовано в текущем релизе; пусто — в roadmap.
Актуальный список реализованных кодов — `gostforge checks`.
