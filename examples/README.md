# Примеры

Папка для примеров использования gostforge. Будет наполнена по мере
реализации фаз.

Планируется добавить:
- `quick-start/` — минимальный пример проверки одной работы из CLI
- `custom-profile/` — пример создания профиля кафедры
- `custom-check/` — пример плагина с пользовательской проверкой
- `api/` — пример вызова из Python-кода
